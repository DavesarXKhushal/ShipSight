Download & Installation

Get ShipSight installed on your computer

Installation Process

  1. Download the installer from our download page or click the button below
  2. Run the installer as an administrator (right-click → "Run as administrator")
  3. Follow the setup wizard - accept the license agreement and choose installation directory
  4. Wait for installation to complete (typically 2-3 minutes)
  5. Launch ShipSight from the desktop shortcut or Start menu
System Requirements: Ensure you have Windows 10 (64-bit) or later, 4GB RAM, and 500MB free disk space.
Pro Tip: Disable antivirus temporarily during installation to avoid false positives with our license protection system.

Camera Configuration

Connect and configure your cameras for optimal recording

Camera Setup Process

  1. Connect your camera via USB, IP network, or HDMI capture device
  2. Open ShipSight Settings → Camera Configuration
  3. Select your camera from the detected devices list
  4. Adjust video settings - resolution, frame rate, and quality
  5. Test the camera using the preview window
  6. Configure recording zones for optimal packing area coverage

Supported Camera Types

USB Webcams

HD, Full HD, 4K cameras

IP Cameras

Network and wireless cameras

CCTV Systems

Existing security cameras

Optimal Camera Positioning

  • Height: 3-5 feet above packing surface
  • Angle: 45-60 degrees for best view
  • Lighting: Ensure adequate lighting for clear video
  • Coverage: Frame the entire packing area
Pro Tip: Use multiple cameras for different angles. ShipSight can record from up to 4 cameras simultaneously.

Ecommerce Platform Integration

Connect ShipSight with your marketplace or ecommerce platform

Integration Setup

  1. Navigate to Settings → Platform Integration
  2. Select your platform from the available options
  3. Enter API credentials or login information
  4. Configure order sync settings - frequency and data fields
  5. Test the connection to ensure proper data flow
  6. Set up automatic order detection for seamless workflow

Platform-Specific Instructions

Amazon Integration
  1. Log into Amazon Seller Central
  2. Go to Settings → User Permissions
  3. Generate API keys for ShipSight
  4. Enter Marketplace ID and Seller ID in ShipSight
Flipkart Integration
  1. Access Flipkart Seller Hub
  2. Navigate to API Management
  3. Create new API application for ShipSight
  4. Copy App ID and Secret to ShipSight settings
Shopify Integration
  1. Install ShipSight app from Shopify App Store
  2. Authorize app permissions
  3. Configure webhook settings
  4. Test order synchronization
API Rate Limits: Each platform has API rate limits. ShipSight automatically manages these to prevent service interruptions.

Your First Recording

Learn how to scan barcodes and create your first shipping video

Recording Workflow

  1. Prepare your packing station - ensure good lighting and camera positioning
  2. Have your barcode scanner ready - USB scanner or smartphone app
  3. Start ShipSight recording mode - click the "Start Recording Session" button
  4. Scan the order barcode - the system will automatically start recording
  5. Pack your items normally - show items clearly to the camera
  6. Scan completion barcode or click "Stop Recording" when finished

Video Tutorial: First Recording

Watch our step-by-step video guide

Best Practices for Recording

  • Show items clearly: Hold items up to camera before packing
  • Demonstrate condition: Show that items are new/undamaged
  • Show packaging materials: Display bubble wrap, padding used
  • Scan shipping label: Include tracking number in video
  • Maintain steady pace: Don't rush - thorough documentation is key
Pro Tip: Add voice commentary describing what you're packing. Audio is recorded along with video for additional evidence.

Managing Your Videos

Organize, search, and access your shipping proof videos

Video Library Features

  1. Access Video Library - click "Video Management" in the main interface
  2. Search by criteria - tracking number, date, customer name, or order ID
  3. Filter recordings - by date range, platform, or recording status
  4. Preview videos - thumbnail view with quick preview option
  5. Export videos - download for dispute resolution or evidence
  6. Share secure links - provide access to specific videos when needed

Using Videos for Dispute Resolution

  1. Identify disputed order using tracking number or order ID
  2. Locate corresponding video in your library
  3. Create dispute evidence package with video and metadata
  4. Submit to marketplace through their dispute resolution process
  5. Provide secure video link if direct upload isn't possible
Pro Tip: Set up automatic cloud backup to ensure your video evidence is always safe and accessible.
Retention Policy: Keep videos for at least 6 months after delivery to cover return windows and dispute periods.
