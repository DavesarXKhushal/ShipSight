System Requirements

Operating System

  • Windows 10 (64-bit) or later
  • Windows 11 (recommended)
  • macOS 11.0+ (coming soon)
  • Linux Ubuntu 20.04+ (coming soon)

Hardware

  • Intel i3 or AMD equivalent
  • 4 GB RAM (8 GB recommended)
  • 500 MB free disk space
  • USB 2.0+ for camera connection

Camera Support

  • USB webcams (HD, 4K)
  • IP/Network cameras
  • RTSP/RTMP streams
  • DirectShow compatible devices

Connectivity

  • Internet connection (for activation)
  • Network access (for IP cameras)
  • USB ports (for barcode scanners)
  • Optional: Bluetooth for mobile scanners
